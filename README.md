# Personal Portfolio Website

A modern and responsive personal portfolio website built using HTML, CSS, and JavaScript. This portfolio showcases my skills, projects, education, certifications, resume, and contact information in a clean and professional design.

## Features

- Responsive design for desktop, tablet, and mobile devices
- Modern user interface with smooth scrolling
- Dark mode toggle
- Animated typing effect
- Project showcase section
- Skills section with progress bars
- Education timeline
- Certificates section
- Animated statistics counters
- Contact form
- Scroll-to-top button
- Resume download option
- Interactive hover effects and animations

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome
- AOS (Animate On Scroll)

## Project Structure

Portfolio/
├── index.html
├── style.css
├── script.js
├── resume.pdf
└── images/

## Purpose

This portfolio was developed to showcase my academic background, technical skills, projects, and achievements as a Computer Engineering student. It serves as a professional online presence for internships, placements, and networking opportunities.
